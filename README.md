## The Challenge

A simple Tic-Tac-Toe program. At the very least this program should:

- allow for a human player
- play against a computer player
- have some user interface (text is fine)
- never lose and win whenever possible


## To Run Application
```
$ cd Tic_Tac_Toe/lib
$ Ruby run.rb
```
## For Test
```
$ cd Tic_Tac_Toe/lib
$ rspec
```
